# PDFs des Cours ETU-Bénin

Ce dossier contient les PDFs des cours pour chaque grade.

## Structure des fichiers :
- `cours-néophyte.pdf` - Cours pour le grade Néophyte
- `cours-constructeur.pdf` - Cours pour le grade Constructeur  
- `cours-navigateur.pdf` - Cours pour le grade Navigateur
- `cours-alchimiste.pdf` - Cours pour le grade Alchimiste

## Instructions :
1. Placez les PDFs correspondants dans ce dossier
2. Nommez-les selon la convention : `cours-{grade}.pdf`
3. Les grades doivent être en minuscules avec des tirets pour les espaces

## Exemple :
- Grade "NÉOPHYTE" → fichier `cours-néophyte.pdf`
- Grade "CONSTRUCTEUR" → fichier `cours-constructeur.pdf`
- Grade "NAVIGATEUR" → fichier `cours-navigateur.pdf`
- Grade "ALCHIMISTE" → fichier `cours-alchimiste.pdf`
